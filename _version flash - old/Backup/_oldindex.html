The Electromagnetic Spectrum - Frequency overview (built by e-builds)

A Flash presentation about the electromagnetic spectrum. It displays all frequencies from 0Hz to Cosmic Radiation. It divides the spectrum in some big regions (such as Radio, microwaves, infrared, visible, ultraviolet, gamma and cosmic) and displays some quick information and some most general uses, sources for that part of the spectrum. Some size comparision with the wavelength can also be seen. This project is under development.
